# J. Гоблины и очереди

## Условие

Гоблины стоят в очереди к шаманам. Есть два типа гоблинов:

1. **Обычные** — встают в **конец** очереди (`+ i`)
2. **Привилегированные** — встают **в середину** очереди (`* i`)
   - При **нечётной** длине очереди встают **сразу за центром**

Нужно обрабатывать запросы и выводить номера гоблинов, которые уходят к шаманам.

---

## Формат ввода

- Первая строка: `N` (1 ≤ N ≤ 10^5) — количество запросов
- Следующие `N` строк содержат запросы:
  - `+ i` — обычный гоблин с номером `i` (1 ≤ i ≤ N) встаёт в конец
  - `* i` — привилегированный гоблин с номером `i` встаёт в середину
  - `-` — первый гоблин уходит к шаманам (очередь не пуста)

## Формат вывода

Для каждого запроса типа `-` выведите номер гоблина, который уходит.

---

**Пример 1**

Ввод:
7

1

3

Вывод:
1
2
