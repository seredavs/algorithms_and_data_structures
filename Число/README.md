# F. Число

## Условие

Вася написал на полоске бумаги большое число. Катя разрезала полоску на несколько частей, в каждой части оказалось одна или несколько идущих подряд цифр.

Теперь Вася не помнит исходное число, но хочет узнать, **какое максимальное число** могло быть написано до разрезания.

Нужно восстановить максимально возможное число, переставив куски в правильном порядке (исходное число — это конкатенация всех кусков в некотором порядке).

---

## Формат ввода

Входной файл содержит одну или более строк (не более 100), каждая строка содержит от 1 до 100 цифр.

Гарантируется, что хотя бы в одной строке первая цифра отлична от нуля.

## Формат вывода

Выведите одну строку — максимальное число, которое могло быть написано.

---

## Примеры

**Пример 1**

Ввод:
2
20
004
66

Вывод:
66220004


**Пример 2**

Ввод:
3
3


Вывод:
3
