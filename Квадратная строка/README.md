# Квадратная строка

## Условие

Строка называется **квадратной**, если она состоит из двух одинаковых половин, записанных дважды подряд.

**Примеры:**
- `aa` → `a` + `a` ✓
- `abcabc` → `abc` + `abc` ✓
- `abab` → `ab` + `ab` ✓
- `aaa` → ✗ (нечетная длина)
- `abcdabc` → ✗ (половины не совпадают)

---

## Формат ввода

- Первая строка: `t` (1 ≤ t ≤ 100) — количество строк
- Следующие `t` строк: строки `s` (1 ≤ |s| ≤ 100) из строчных латинских букв

## Формат вывода

Для каждой строки выведите:
- `YES` — если строка квадратная
- `NO` — иначе

---

**Ввод:**
10
a
