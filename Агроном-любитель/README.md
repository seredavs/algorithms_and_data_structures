# A. Агроном-любитель

## Условие

Лёша посадил вдоль грядки `n` цветков. Каждый цветок имеет свой вид — целое число из каталога.

Лёша хочет сфотографировать отрезок из одного или нескольких подряд идущих цветков. Фотография будет некрасивой, если на ней есть **три одинаковых цветка подряд**.

Нужно найти **самый длинный** участок грядки, на котором **нет трех цветков одного вида подряд**.

---

## Формат ввода

- Первая строка: `n` (1 ≤ n ≤ 200 000) — количество цветков
- Вторая строка: `n` целых чисел `a_i` (1 ≤ a_i ≤ 10^9) — виды цветков

## Формат вывода

Выведите два числа:
- Номер первого цветка на участке
- Номер последнего цветка на участке

Цветки нумеруются с **1** до `n`.

> Если самых длинных участков несколько, выведите тот, который **начинается раньше**.

---

## Пример

**Ввод:**
6
5 6 6 6 23 9

text

**Вывод:**
3 6
