# I. Машинки

## Условие

У Пети `N` различных машинок, которые хранятся на полке. Одновременно на полу может находиться не более `K` машинок.

Петя играет с одной машинкой на полу. Если он хочет поиграть с другой машинкой:
- Если машинка **на полу** — он берет её сам
- Если машинка **на полке** — он просит маму

Мама может:
1. Достать нужную машинку с полки
2. Одновременно поставить **любую** машинку с пола на полку (если на полу уже `K` машинок)

Мама знает последовательность, в которой Петя будет играть. Нужно минимизировать количество операций по подъёму машинок с пола.

**Изначально все машинки на полке.**

---

## Формат ввода

- Первая строка: `N`, `K`, `P` (1 ≤ K, N ≤ 100 000, 1 ≤ P ≤ 500 000)
- Следующие `P` строк: номера машинок в порядке игры

## Формат вывода

Одно число — минимальное количество операций мамы.

---

## Пример

**Ввод:**
3 2 7
1
2
3
1
3
1
2

**Вывод:**
4

**Пояснение:**
1. Снять машинку 1
2. Снять машинку 2
3. Поднять машинку 2 и снять машинку 3
4. Поднять машинку 3 (или 1) и снять машинку 2
