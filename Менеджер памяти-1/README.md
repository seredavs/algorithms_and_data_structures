# K. Менеджер памяти-1

## Условие

Менеджер управляет массивом из `N` ячеек памяти (пронумерованы от 1 до N). Нужно обрабатывать запросы:

1. **Выделение памяти** (`+K` или просто `K`):
   - Найти свободный блок из `K` **последовательных** ячеек
   - Блок должен начинаться **сразу после занятой ячейки** (или с начала массива)
   - Если есть несколько подходящих блоков — берётся первый подходящий
   - При успехе — ячейки становятся занятыми
   - При неудаче — запрос отклоняется

2. **Освобождение памяти** (`-T`):
   - Освободить память, выделенную в запросе с номером `T`
   - Гарантируется, что запрос `T` — это запрос на выделение и ранее не освобождался
   - Если запрос `T` был отклонён — текущий запрос игнорируется

---

## Формат ввода

- Первая строка: `N` (1 ≤ N ≤ 2^31 − 1) и `M` (1 ≤ M ≤ 10^5) — количество ячеек и запросов
- Следующие `M` строк:
  - `K` (> 0) — запрос на выделение `K` ячеек (1 ≤ K ≤ N)
  - `-T` (< 0) — запрос на освобождение памяти запроса `T` (1 ≤ T < i)

## Формат вывода

Для каждого запроса на выделение памяти выведите:
- Номер первой ячейки выделенного блока — при успехе
- `-1` — при отклонении
