# E. Коровы в стойла

## Условие

На прямой расположены `N` стойл с известными координатами. Нужно расставить `K` коров по стойлам так, чтобы **минимальное расстояние между любыми двумя коровами** было как можно больше.

Требуется найти максимально возможное значение этого минимального расстояния.

---

## Формат ввода

- Первая строка: `N` (2 < N ≤ 10^5) — количество стойл, `K` (1 < K < N) — количество коров
- Вторая строка: `N` натуральных чисел в порядке возрастания — координаты стойл (координаты ≤ 10^9)

## Формат вывода

Одно число — наибольшее возможное минимальное расстояние между коровами.

---

## Примеры

**Пример 1**

Ввод:
6 3
2 5 7 11 15 20

Вывод:
9


**Пример 2**

Ввод:
5 3
1 2 3 100 1000

Вывод:
99
