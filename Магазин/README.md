# H. Магазин

## Условие

В магазине акция: **каждый k-й товар бесплатно**. 

В чеке из `n` товаров бесплатно достаются `⌊n/k⌋` самых дешёвых товаров.

Билл может **разбить товары на несколько чеков**, чтобы заплатить меньше.

Нужно найти минимальную сумму, которую Билл сможет заплатить.

---

## Формат ввода

- Первая строка: `n` (1 ≤ n ≤ 100 000) — количество товаров, `k` (2 ≤ k ≤ 100) — параметр акции
- Вторая строка: `n` целых чисел `a_i` (1 ≤ a_i ≤ 10 000) — цены товаров

## Формат вывода

Одно число — минимальная сумма, которую Билл сможет заплатить.

---

## Пример

**Ввод:**
5 2
200 100 1000 400 100


**Вывод:**
1300
