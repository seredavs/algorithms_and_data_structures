# D. Профессор Хаос

## Условие

В начале первого дня у профессора Хаоса есть `a` бактерий. Каждый день происходит следующее:

1. Все бактерии помещаются в инкубатор, где каждая бактерия превращается в `b` новых бактерий
2. После инкубатора `c` бактерий забирают для опытов и уничтожают
   - Если бактерий меньше `c`, забирают всех, и эксперимент **заканчивается**
3. Оставшиеся бактерии помещают в контейнер
   - Если их больше `d`, в контейнер помещают только `d` бактерий, остальные уничтожают

Нужно определить, сколько бактерий будет в контейнере **после `k`-го дня**.

---

## Формат ввода

Единственная строка содержит пять целых чисел:
- `a` (1 ≤ a ≤ 1000) — начальное количество бактерий
- `b` (1 ≤ b ≤ 1000) — во сколько раз увеличивается количество
- `c` (0 ≤ c ≤ 1000) — сколько забирают на опыты
- `d` (1 ≤ d ≤ 1000, a ≤ d) — максимальная вместимость контейнера
- `k` (1 ≤ k ≤ 10^18) — количество дней

## Формат вывода

Одно число — количество бактерий после `k`-го дня.

Если эксперимент завершился в `k`-й день или раньше, выведите `0`.

---

## Примеры

**Пример 1**

Ввод:
1 3 1 5 2


Вывод:
5


**Пример 2**

Ввод:
1 2 0 4 3


Вывод:
4


**Пример 3**

Ввод:
1 2 3 5 2


Вывод:
0
